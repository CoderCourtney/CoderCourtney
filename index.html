PORTFOLIO

Restaurant Reservations

Periodic Tables: an app for restaurant employees to handle phone reservation booking as well as giving them the ability to seat a table, search for a reservation and complete reservations. This app was built utilizing React.js, React hooks, Express, Knex, REST APIs, Bootstrap & Jest.

Decoder Ring Project

Complex algorithms were written in JavaScript to encode and decode all kinds of fun messages! A fun app to make some treasure hunt or secret clubhouse messages! I also wrote a series of unit tests using Mocha and Chai. It includes HTML and CSS.

Pomodoro Timer

This is a dual productivity timer, built with React components rendering consecutive timers. It helps with focus when getting a project done. The user can set the timer for how long they want to work and the length of their breaks. The tech stack used includes React.js, HTML, CSS, Bootstrap and NPM.
